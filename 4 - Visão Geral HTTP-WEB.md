Nesse módulo faremos uma introdução a HTTP/WEB demonstrando uma visibilidade de como funciona o tráfego entre o cliente em seu Web Browser e um servidor Web hospedado em um server

# Overview WEB/HTTP

**HTTP:** Protocolo que fornece a comunicação entre o cliente no Browser com o Webserver
![[Pasted image 20251109184133.png]]

Fluxo da conexão: Cliente (Browser) ←→ Servidor (WebServer)

Ex: Firefox ←→ Apache em server Linux

Outro ponto importante que revisaremos em outros momentos é que o HTML funciona com o conceito de **HEADER e BODY** suportando operações como **GET,POST,DELETE,OPTIONS,PUT** e outras.

Códigos de status no HTTPS:

- **200: OK**
- **301: Redirecionamento**
- **403: Acesso bloqueado**
- **404: Not found**
- **500: Erro do server**
