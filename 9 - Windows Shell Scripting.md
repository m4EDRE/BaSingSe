A sessão a seguir se trata de uma aprofundamento na construição de scripts powershell *(.ps1)*. Que podem ser úteis no dia a dia de um Pentester

# Introdução ao PowerShell

Para criar um script em powershell primeiro precisamos abrir um txt com final **.ps1** usando o comando notepad

`notepad primeiroscript.ps1`

- Comentários - #
    
- ./primeiroscript.ps1 - Executa script

- get-location - pega localização (pwd)
